# Data Science with Python

A hands-on data science project repository built by following the Data Science from Free Code Camp

## 📚 What's Covered

This repo walks through the full beginner-to-intermediate data science
pipeline using Python, including:

- **Python Basics** – Core syntax and programming fundamentals
- **Pandas & NumPy** – Data manipulation and numerical computing
- **Web Scraping** – Extracting data from the web using Pandas
- **Filtering & Data Extraction** – Querying and slicing datasets
- **Reshaping & Pivoting DataFrames** – Restructuring data for analysis
- **Data Visualization** – Creating charts and graphs
- **GroupBy & Aggregation** – Summarizing data by groups
- **Merging & Concatenating DataFrames** – Combining datasets
- **Regular Expressions** – Pattern matching in text data
- **Data Cleaning** – Handling messy, real-world data
- **Machine Learning with Python** – Intro to ML concepts
- **Text Classification** – Building a classifier with scikit-learn

## 🗂️ Projects

| # | Project | Topics |
|---|---------|--------|
| 1 | Web Scraping with Pandas | Data collection, HTML parsing |
| 2 | Data Visualizations | Matplotlib, charts, trends |
| 3 | Data Cleaning with Pandas | Handling nulls, inconsistencies |
| 4 | Text Classification with scikit-learn | NLP, ML pipeline |

## 🛠️ Tech Stack

- Python 3
- Pandas
- NumPy
- Matplotlib
- scikit-learn
- Jupyter Notebook

## 🚀 Getting Started

```bash
git clone https://github.com/sleepsonrockss/Data-Science.git
cd Data-Science
pip install -r requirements.txt
jupyter notebook
```
